Node.js Vite AWS S3 CI/CD Jenkins Pipeline

About this project

This repo contains a Vite React frontend and a Node.js backend example. The Jenkins pipeline builds the project and deploys the static `dist/` folder to an S3 bucket. Ideal for showcasing CI/CD and modern frontend toolchain on your resume.

Quick commands

# install & run (frontend)
npm ci && npm run build && npm start

Pipeline

Jenkinsfile builds with NodeJS tool, runs `npm run build`, then uses AWS CLI to sync `dist/` to S3. Credentials are provided via Jenkins Credentials Binding.

Permissions

Ensure the IAM user used by Jenkins has S3 put/delete access on the target bucket.
